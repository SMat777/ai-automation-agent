Document Analysis

Detects document type, extracts emails, dates, URLs, organizations, heading structure, and key points. Works on any text — emails, reports, markdown, tables.

Data Extraction

Pulls structured key-value pairs from unstructured text. Supports three strategies: key-value parsing, markdown table extraction, and list parsing. Auto mode tries all three.

Text Summarization

Condenses long text into key takeaways. Uses Claude AI when an API key is configured, otherwise falls back to extractive summarization — both are fully functional.

Data Pipeline

Runs a full TypeScript automation workflow: Fetch data from a live API → Clean → Filter → Aggregate → Format. Each step is composable and type-safe via Zod validation.

How it works

🐍

Python Agent

ReAct decision loop with Claude API. Reasons about tasks, selects tools, synthesizes results.

📘

TypeScript Pipeline

Composable data transformations with Zod validation. Fetch, clean, filter, aggregate, format.

⚡

FastAPI Server

REST API with Pydantic validation. Serves the web UI and exposes all tools as endpoints.

🔗

Integration

Agent triggers pipelines at runtime. Pipeline results feed back for AI analysis and reporting.
