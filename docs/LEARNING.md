# Learning Journal

A chronological record of what I learned, what was challenging, and what clicked during development.

## How to Use This Document

Each entry follows this format:
- **Date** and topic
- **What I learned** — the key concept or technique
- **What was challenging** — where I got stuck
- **What clicked** — the "aha" moment

## Entries

### 2026-04-16 — Project Setup

**What I learned:** Setting up a polyglot project (Python + TypeScript) with shared CI/CD. GitHub Actions can run multiple jobs in parallel with different language setups.

**What was challenging:** Deciding on project structure — where to draw the line between agent and automation responsibilities.

**What clicked:** The agent handles *reasoning* (what to do), the pipeline handles *execution* (how to do it). Clean separation.
